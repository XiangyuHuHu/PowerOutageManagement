# 快速测试新功能指南

## 🚀 推荐测试方式：H5浏览器

### 步骤1：启动后端服务
```bash
cd web-backend
python server.py
```

### 步骤2：在HBuilderX中运行
1. 打开 `miniprogram` 文件夹
2. 点击 `运行` → `运行到浏览器` → `Chrome`
3. 等待编译完成

### 步骤3：测试新功能

#### ✅ 测试注册功能
1. 访问登录页面
2. 点击 `注册新用户` 按钮
3. 填写注册信息：
   - 用户名：testuser
   - 密码：123456
   - 确认密码：123456
   - 真实姓名：测试用户
   - 角色：电工
4. 点击注册，应该跳转到登录页面

#### ✅ 测试管理员功能
1. 使用管理员账号登录：
   - 用户名：admin
   - 密码：admin
2. 进入管理员页面，应该看到4个功能卡片：
   - 审批管理
   - 系统统计
   - 用户管理
   - 检修管理

#### ✅ 测试页面跳转
1. 点击 `用户管理` → 应该跳转到用户管理页面
2. 点击 `系统统计` → 应该跳转到统计页面
3. 点击 `检修管理` → 应该跳转到检修页面
4. 各页面的返回按钮应该正常工作

## 🔧 如果微信开发者工具登录失败

### 解决方案：
1. **重新登录微信开发者工具**
   - 点击右上角头像
   - 重新扫码登录

2. **清除缓存**
   ```bash
   # macOS
   rm -rf ~/Library/Application\ Support/微信web开发者工具/
   ```

3. **使用H5浏览器替代**
   - 在HBuilderX中选择 `运行到浏览器`
   - 避免微信开发者工具的登录问题

## 🚨 故障排除

### 如果页面无法访问：
1. **检查后端服务**
   ```bash
   curl http://localhost:5050/api/login
   ```

2. **检查编译状态**
   - 确认HBuilderX编译成功
   - 没有CSS文件缺失错误

3. **清除浏览器缓存**
   - 按 Ctrl+Shift+R 强制刷新
   - 或清除浏览器缓存

### 如果API请求失败：
1. **确认后端运行**
   ```bash
   cd web-backend
   python server.py
   ```

2. **检查网络连接**
   ```bash
   curl http://localhost:5050/api/users
   ```
