# 快速测试指南

## 🚀 一键启动测试

### 1. 启动Web后端
```bash
cd web-backend
python server.py
```

### 2. 启动H5项目
在HBuilderX中：
1. 打开 `miniprogram` 文件夹
2. 点击 `运行` → `运行到浏览器` → `Chrome`
3. 等待编译完成

## 🔍 故障排除

### 如果看不到新页面：
1. **强制重新编译**
   ```
   运行 → 重新编译
   ```

2. **清除缓存**
   ```
   运行 → 清除缓存并运行
   ```

3. **检查文件路径**
   ```
   确认以下文件存在：
   - pages/register/register.vue
   - pages/user_management/user_management.vue
   - pages/stats/stats.vue
   - pages/repair/repair.vue
   ```

### 如果API请求失败：
1. **检查后端服务**
   ```bash
   # 确保后端正在运行
   curl http://localhost:5050/api/login
   ```

2. **检查网络连接**
   ```bash
   # 测试API连接
   curl -X POST http://localhost:5050/api/login \
     -H "Content-Type: application/json" \
     -d '{"username":"admin","password":"admin"}'
   ```
